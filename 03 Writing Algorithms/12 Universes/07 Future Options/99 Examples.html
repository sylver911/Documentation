Example 1: 30-day Expiring Straddle

The following example selects an ATM call and an ATM put contract with at least 30 days from expiry to form a straddle.

public override void Initialize()
{
    var future = AddFuture("ES");
    var futureOption = AddFutureOption(future.Symbol);
    futureOption.SetFilter(optionFilterUniverse => optionFilterUniverse.IncludeWeeklys().Straddle(30));
}
def initialize(self):
    future = self.add_future("ES")
    future_option = self.add_future_option(future.symbol)
    future_option.set_filter(lambda option_filter_universe: option_filter_universe.include_weeklys().straddle(30))
